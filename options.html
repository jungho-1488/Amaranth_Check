인증 관리
Google Cloud Console 설정 방법
  1. Google Cloud Console → 프로젝트 생성 (또는 기존 프로젝트 선택)
  2. API 및 서비스 → 라이브러리에서 Google Sheets API 활성화
  3. 사용자 인증 정보 → OAuth 클라이언트 ID 만들기
  4. 애플리케이션 유형: 웹 애플리케이션 선택
  5. 승인된 리디렉션 URI에 위의 Redirect URI 추가
  6. 클라이언트 ID와 시크릿을 위에 입력 후 저장
  7. Google 인증 버튼으로 최초 로그인 진행
